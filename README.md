#  Portfolio 

Moja osobista strona internetowa typu One-Page, pełniąca rolę cyfrowej wizytówki i portfolio. Projekt został stworzony od podstaw, aby w estetyczny i przejrzysty sposób zaprezentować moje umiejętności programistyczne, ścieżkę rozwoju oraz zrealizowane aplikacje.

Strona została zaprojektowana z myślą o nowoczesnych trendach UI (ciemny motyw z neonowymi akcentami) oraz pełnej responsywności na urządzeniach mobilnych.

##  Funkcje i Możliwości

* **Responsive Web Design (RWD):** Układ strony automatycznie dostosowuje się do rozmiaru ekranu (wykorzystanie CSS Grid, Flexbox oraz zapytań `@media`), zapewniając idealny wygląd zarówno na monitorach, jak i smartfonach.
* **Nowoczesny Design (Dark Mode):** Interfejs oparty na ciemnej palecie barw z gradientami i czytelnymi, jaskrawymi akcentami, co nadaje stronie profesjonalny, "hakerski" charakter.
* **Płynna Nawigacja (Smooth Scrolling):** Przyklejone do góry menu (Sticky Nav) pozwala na szybkie i płynne przemieszczanie się między sekcjami strony (`#bio`, `#projekty`, `#kontakt`).
* **Interaktywny Kontakt (Ochrona przed spamem):** Adres e-mail jest domyślnie ukryty pod przyciskiem i ujawnia się dopiero po interakcji użytkownika (skrypt JS), co chroni skrzynkę przed botami zbierającymi dane.
* **Silnik Galerii (JS):** Wbudowany, lekki skrypt JavaScript przygotowany pod dynamiczne ładowanie i przewijanie wielu zrzutów ekranu dla każdego projektu (Image Slider).


##  Technologie

* **HTML5** (Semantyczna struktura, integracja ikon Devicon)
* **CSS3** (CSS Grid, Flexbox, Transitions, zmienne, Sticky Positioning)
* **JavaScript (ES6+)** (Manipulacja DOM, obsługa zdarzeń, dynamiczne style)

##  Struktura projektu

* `index.html` – Główny plik ze strukturą strony, podzielony na logiczne sekcje (O mnie, Umiejętności, Projekty, Kontakt).
* `style.css` – Zaawansowany arkusz stylów odpowiedzialny za kolorystykę, ułożenie elementów oraz responsywność (Media Queries).
* `script.js` – Logika interaktywnych elementów (m.in. bezpieczne wyświetlanie adresu e-mail oraz system obsługi galerii zdjęć).
